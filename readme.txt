# Portofolio Web Haidi Fathur Rahman

## Deskripsi
Ini adalah proyek portofolio web elegan dan profesional untuk Haidi Fathur Rahman, seorang spesialis GIS dan pengembang solusi. Portofolio ini dirancang untuk menampilkan keahlian, pengalaman, dan proyek-proyek dalam bentuk yang menarik dan responsif.

## Fitur
- **Desain Responsif**: Tampilan yang optimal di semua perangkat (desktop, tablet, smartphone)
- **Mode Gelap/Terang**: Toggle antara tema gelap dan terang sesuai preferensi pengguna
- **Bilingual**: Dukungan bahasa Indonesia dan Inggris
- **Animasi Elegan**: Efek visual yang halus untuk pengalaman pengguna yang lebih baik
- **Struktur Modular**: Organisasi kode yang rapi dan terstruktur

## Teknologi yang Digunakan
- HTML5
- CSS3 (dengan variabel CSS untuk tema)
- JavaScript (vanilla, tanpa framework)
- Font Awesome untuk ikon
- Google Fonts untuk tipografi

## Halaman
1. **Beranda (index.html)**
   - Hero section dengan pengenalan singkat
   - Highlight keahlian utama
   - Preview proyek terbaru
   - Pengalaman kerja singkat
   - Call-to-action untuk kontak

2. **Tentang (about.html)**
   - Informasi personal dan profesional
   - Timeline perjalanan karir
   - Keahlian teknis dengan visualisasi level

3. **Pendidikan (education.html)**
   - Riwayat pendidikan formal
   - Sertifikasi dan pelatihan
   - Pencapaian akademis

4. **Pengalaman (experience.html)**
   - Riwayat pekerjaan dengan detail
   - Pencapaian di setiap posisi
   - Proyek penting yang dikerjakan

5. **Proyek (projects.html)**
   - Galeri proyek dengan kategori
   - Detail untuk setiap proyek
   - Filter berdasarkan jenis keahlian

6. **Kontak (contact.html)**
   - Formulir kontak
   - Informasi kontak langsung
   - Link ke platform freelance (Fastwork & Upwork)
